# Leistungskurve 

Dieses Projekt lädt Aktivitätsdaten aus einer CSV-Datei, sortiert die Leistungswerte mit einem eigenen Sortieralgorithmus (Bubble Sort) und erzeugt daraus eine einfache Power-Curve-Grafik.
